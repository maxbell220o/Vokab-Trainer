# Vokab-Trainer
Ist ein eifacher Vokabeltrainer. Nicht Schön, aber Gut.

## Instalier Anleitung

1. Herunterladen Des Repos
2. Produckt-Key anfragen (boeden.versorgung.6f@icloud.com) (Code ID: JANUAR)
3. Program starten und code eingeben.

## Weitere Infos:
- Volständig Kostenlos (Produckt Key Prüft nur das das Produkt nicht in Falsche Hände gerät) 
- Die App ist kein Auslauf Produkt.
- Bei Beschwerden oder Verbesserungsvorschlägen melden sie sich unter boeden.versorgung.6f@icloud.com.


## Verion 1:

### Funtionen:
- Merere Fächer(Englisch,Latein)
- Fachsystem 1-5
- Widerhole der Falschen Vokabeln
